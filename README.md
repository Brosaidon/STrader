## Introduction

StarTrader is a web app where you take the roll of a traveling trader. You need manage a ship with crew, buy whares at a profitable price and travle to another location to sell your wares for a profit.

StarTrader is a full stack demo application that aims to show different layers of software concept in an interesting and interactive way. My goal is to introduce different software in a way that is easy to understand and hopefully easy to read and understand.
